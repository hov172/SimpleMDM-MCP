# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating — filing issues, opening PRs, or engaging in discussions — you agree to uphold this code in all project spaces (this repository, related discussion channels, and any community space where you represent the project).

## Reporting

Concerns about behavior in this project's spaces can be reported privately to:

- **GitHub:** DM [@hov172](https://github.com/hov172)
- **MacAdmins Slack:** DM [@hov172](https://macadmins.slack.com)

All reports are reviewed and handled confidentially. Maintainers are obligated to respect the privacy and security of the reporter.

## Enforcement

Project maintainers may take any action they deem appropriate, including warnings, temporary bans, or permanent removal from project spaces, in response to behavior that violates this Code of Conduct. Enforcement decisions follow the guidelines outlined in the Contributor Covenant linked above.

For security-related concerns (vulnerabilities, credential exposure), see [SECURITY.md](SECURITY.md) instead — security has a dedicated private disclosure path.
